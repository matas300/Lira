# Examples - Commercialista Fiscale

## Example 1 - Netto da fatturato forfettario
### User request
Calcolami quanto mi resta in tasca con 68.000 euro di ricavi, coefficiente 67%, Gestione Separata, anno 2026.

### Expected behavior
- classifica il caso come simulazione forfettario;
- espone i dati noti e le ipotesi;
- calcola reddito lordo forfettario;
- separa contributi e imposta;
- mostra netto annuo e netto mensile medio;
- chiude con punti da verificare.

## Example 2 - Confronto forfettario vs ordinario
### User request
Con 75.000 euro di ricavi mi conviene il forfettario o l'ordinario?

### Expected behavior
- classifica come confronto;
- costruisce almeno 2 scenari coerenti;
- usa stessa base ricavi per entrambi;
- confronta netto, carico totale e semplicità gestionale;
- non dà una risposta vaga tipo "dipende" senza numeri.

## Example 3 - Verifica accesso/permanenza
### User request
Posso stare nel forfettario se l'anno scorso avevo anche redditi da lavoro dipendente e un collaboratore?

### Expected behavior
- classifica come verifica accesso/permanenza;
- elenca i parametri da controllare;
- non inventa il responso se mancano dati;
- struttura la risposta come checklist di requisiti.

## Example 4 - Fatturato necessario per un netto target
### User request
Quanto devo fatturare per avere 3.000 euro netti al mese?

### Expected behavior
- classifica come fatturato da netto desiderato;
- chiarisce il regime e la gestione previdenziale necessari per fare il conto;
- se mancano dati, usa ipotesi dichiarate;
- restituisce il fatturato stimato annuo e mensile con formule leggibili.
